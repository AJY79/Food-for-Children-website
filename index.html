As there are 2 weeks in the Easter holidays we will be providing 4 meals for each child, breakfast for each weekday, squash/Juice and hopefully an Easter Egg for each child. So if you're in a position to help with any of these then that would be truly amazing and you can find a list of our current requests either on the Amazon link or further down this post. Drop off Point is 1 Whiteley Avenue, Totnes, TQ9 5FQ or alternatively message us to arrange collection. Thank you so much for your help with this.
